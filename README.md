# PK-Pokedex
This is a web-app for quickly accessing documentation for MrKeating's Platinum Kaizo romhack. I have aggregated the data available on the mastersheet's provided and utilized Claude's CLI agent to speed up development tremendously.

The user is able to quickly search for any Pokemon and easily identify learnsets, encounter tables, evolution requirements, and base stats.

This is an alternative to the official documentation, which can be difficult to navigate at-times.

## Assets
Contains the information pulled from PokeAPI. 

## Docs
Information from MrKeating's official Platinum Kaizo documentation. The encounters.csv file had to be mutated so the data could be more easily interpreted. 

<br>
Built with ReactJS and Vite.

Thank you to the PokeAPI Restful API v2 for allowing me to quickly query and save images of the Pokemon, berry, type, and held-item sprites for assets.
